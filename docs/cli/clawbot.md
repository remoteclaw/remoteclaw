# `remoteclaw clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `remoteclaw clawbot qr` (same behavior as [`remoteclaw qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `remoteclaw clawbot qr` -> `remoteclaw qr`
