# `remoteclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
remoteclaw tui
remoteclaw tui --url ws://127.0.0.1:18789 --token <token>
remoteclaw tui --session main --deliver
```
